Array

Array é uma variável que pode armazenar mais de um valor.

Os valores são armazenados em posições, chamadas de índices.

Os índices começam em 0.

Para criar um array, usamos a função array().

Exemplo:

        $frutas = array('maçã', 'banana', 'laranja');

Para acessar um valor do array, usamos o índice correspondente.

Exemplo:

        echo $frutas[0]; // maçã
        echo $frutas[1]; // banana
        echo $frutas[2]; // laranja

Para adicionar um valor ao array, usamos a função array_push().

Exemplo:

        array_push($frutas, 'uva');

Para remover um valor do array, usamos a função array_pop().

Exemplo:

        array_pop($frutas);

Para saber o tamanho do array, usamos a função count().

Exemplo:

        echo count($frutas); // 3

Para percorrer um array, usamos o laço de repetição foreach().

Exemplo:

        foreach ($frutas as $fruta) {
            echo $fruta;
        }

Para percorrer um array associativo, usamos o laço de repetição foreach().

Exemplo:

        $pessoa = array('nome' => 'João', 'idade' => 30);
        foreach ($pessoa as $chave => $valor) {
            echo $chave . ': ' . $valor;
        }

Para ordenar um array, usamos a função sort().
